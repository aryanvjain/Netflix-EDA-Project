# Netflix Content Analysis using Python

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Titles dataset to understand content distribution, release patterns, genres, ratings, and other insights using Python.

## Tools Used
Python
Pandas
NumPy
Matplotlib
Google Colab

## Dataset
Netflix Titles Dataset from Kaggle

## Objectives
Clean and preprocess the dataset
Handle missing values
Perform feature engineering
Explore trends using visualizations
Draw meaningful insights from the data

## Questions Answered
Movies vs TV Shows
Top contributing countries
Most common genres
Release trends over the years
Monthly content additions
Rating distribution
Movie duration distribution
Average movie duration

## Key Findings
### Conclusion
This project gave me hands-on experience with the complete Exploratory Data Analysis (EDA) process using the Netflix Titles dataset. Starting with raw data, I cleaned the dataset by handling missing values, removing unnecessary columns, standardizing text, converting data types, and creating new features like **Year Added** and **Month Added** to make the analysis more meaningful.

Through this analysis, I found that Movies make up a much larger portion of Netflix's catalog than TV Shows. The United States and India contribute the highest number of titles, while genres like **International Movies**, **Dramas**, and **Comedies** are among the most common. I also observed that Netflix added the highest number of titles during **July** and **December**. Although this could be related to holiday seasons when people generally have more free time, the dataset doesn't contain enough information to confirm this. Most movies also fall within the standard feature-length duration, averaging around **100 minutes**.

One of the biggest lessons from this project was understanding the importance of making conclusions based only on what the data actually tells us. Since this dataset contains only Netflix's content catalog and not user viewing data, I couldn't make claims about what viewers prefer or what content is the most popular. Instead, I focused on identifying patterns and supporting them with evidence from the data.

Overall, this project helped me become more comfortable with **Pandas**, **Matplotlib**, and **Seaborn**, while also improving my understanding of data cleaning, feature engineering, visualization, and drawing meaningful insights from real-world data. As my first end-to-end EDA project, it has given me a strong foundation for more advanced data analysis and machine learning projects in the future.

## Limitations
Dataset does not contain user viewing data.
Cannot determine popularity or viewer preferences.
